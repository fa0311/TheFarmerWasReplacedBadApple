# TheFarmerWasReplacedBadApple

`TheFarmerWasReplaced` is a program that plays *Bad Apple!!* within the game. To run on a restricted Python runtime, it includes some intentionally verbose processing. Because I/O and program file size are limited, it employs advanced optimizations and clever techniques.

```sh
git clone https://github.com/fa0311/TheFarmerWasReplacedBadApple
cd TheFarmerWasReplacedBadApple
pip install -r requirements.txt
./install.ps1
python main.py
```
